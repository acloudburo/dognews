Code Syntax Highlighting

Posted on Apr 6

This note show some examples for Code Highlighting in the Cloudburo  Curation Platform.
The support in Evernote is very limited when it comes to Code Samples. The Curation Platform tries it best to convert the note code samples into meaningful and nicely highlighted text.

First Test: Ruby
<code> ruby   helpers do    def table_of_contents(resource)        content = File.read(resource.source_file)        toc_renderer = Redcarpet::Render::HTML_TOC.new       markdown = Redcarpet::Markdown.new(toc_renderer, nesting_level: 2)       # nesting_level is optional       markdown.render(content)   end end </code>
Second Test: Java
<code> java protected void processExtractMedia(org.jsoup.nodes.Document doc , String prependSpaces , String mime , String noteGuid) throws Exception {   Elements elems = doc .select( “en-media[type$=” + mime + “]” );   Iterator<Element> it = elems .iterator();
  while ( it .hasNext()) {
  Element elem = it .next();
  String hash = elem .attr( “hash” );
  String img = “ imgx111_” + hash + “.” + mime + “ ” ;
  logger .debug( “Extend en-note with image tag ” + img );
  elem .prepend( img );
  Resource res   = noteStore .getResourceByHash( noteGuid , hexStringToByteArray( hash ), true , false , false );
  extractMediaResource( res , imageDirectory ,  hash );
  } } </code>
